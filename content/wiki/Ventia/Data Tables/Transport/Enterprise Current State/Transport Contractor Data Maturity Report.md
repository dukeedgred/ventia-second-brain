# Transport Contractor Data Maturity Report

This page links to the interactive HTML report comparing documented Transport contract/schema contexts by data maturity and analytics sophistication. It now also includes common-analysis coverage, missing-contractor visibility, score rationale/missing-evidence notes, and a per-table inferred-use section for contractor/schema tables.

## Interactive Report

[Open the interactive contractor data-maturity report](transport-contractor-data-maturity-report.html)

## Evidence Boundary

The report uses current Transport table documentation, Databricks metadata exported during the 2026-06-09 table refresh, and curated Transport wiki notes. It excludes Maximo from the scored comparison for now.

The scoring is metadata-based. It does not certify row-level data quality, production readiness, refresh reliability, or actual dashboard usage.

The table-use section is explicitly inferential. It uses documented domain/purpose fields, table names, column names, and view SQL where available, and labels each row with confidence and evidence basis rather than writing inferred facts back into the source table pages.
